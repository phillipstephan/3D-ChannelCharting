# Scenario 1: Factory Hall

## How to Use the Code
The Jupyter notebooks in this repository are numbered.
You must execute them in the right order.

* `0_Dataset_RT.ipynb`: Use [Sionna RT](https://nvlabs.github.io/sionna/rt/index.html) and the 3D model provided in the `dataset` folder to create a CSI dataset. Due to the large file size, the dataset itself is not included in this git repository.
* `1_DelaySpread.ipynb`: Compute the delay spread for each antenna array and for each datapoint.
* `2_AoA_Estimation.ipynb`: Estimate AoAs for each antenna array and for each datapoint.
* `3_ClassicalLocalization.ipynb`: Estimate UE positions based on estimated AoAs and triangulation.
* `4_DissimilarityMatrix.ipynb`: Compute the dissimilarity matrix based on the angle-delay profile (ADP) dissimilarity metric. Learn more about it in [our tutorial](https://dichasus.inue.uni-stuttgart.de/tutorials/tutorial/dissimilarity-metric-channelcharting/).
* `5_ChannelCharting.ipynb`: Train the neural network that implements the forward charting function. Training is based on a Siamese neural network and the computed dissimilarity matrix.
* `6_AugmentedChannelCharting.ipynb`: Combine channel charting and classical localization for precise, absolute UE localization.

The notebooks use `dataset_RT.py` to load the TFRecord dataset generated by `0_Dataset_RT.ipynb`, and later notebooks also expect the intermediate `.npy` files written to the `dataset` and `results` folders to exist.
